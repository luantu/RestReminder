# RestReminder

一个macOS应用，用于定时全屏提醒休息，保护你的眼睛和身体健康。

## 功能特点

1. **全屏提醒**：支持多屏幕，每个屏幕都会显示提醒
2. **自定义提醒周期**：可以设置任意分钟后提醒
3. **美观的提醒界面**：使用桌面背景作为提醒背景，半透明遮罩
4. **智能屏蔽**：支持配置屏蔽的应用，当特定应用在前台运行时不提醒
5. **状态栏倒计时**：在菜单栏显示剩余时间
6. **丰富的菜单功能**：支持停止计时、重新计时、暂停计时、退出等

## 支持的屏蔽应用

默认屏蔽以下应用：
- Keynote
- Microsoft Teams
- Zoom
- FaceTime
- Slack

## 安装和运行

### 从源代码构建

```bash
git clone <repository-url>
cd RestReminder
swift build
swift run
```

### 直接运行

```bash
swift run
```

## 使用说明

1. 应用启动后会在菜单栏显示一个时钟图标和倒计时
2. 点击菜单栏图标可以打开设置窗口
3. 在设置窗口中可以：
   - 调整提醒间隔（分钟）
   - 查看屏蔽的应用列表
   - 重置计时
   - 暂停/继续计时
   - 退出应用
4. 当倒计时结束时，所有屏幕会显示全屏提醒
5. 点击提醒窗口或"知道了"按钮可以关闭提醒并重新开始计时

## 技术实现

- 基于Swift和SwiftUI开发
- 使用AppKit框架实现状态栏和窗口管理
- 使用Combine框架进行状态管理
- 支持macOS 13.0及以上版本

## 权限说明

应用需要以下权限：
- AppleScript访问权限：用于获取桌面背景
- 辅助功能访问权限：用于检测前台应用

## 自定义设置

当前版本的设置是硬编码的，你可以通过修改`Settings`结构体来自定义：

```swift
struct Settings {
    var reminderInterval: Int = 30 // 分钟
    var blockedApps: [String] = [
        "com.apple.Keynote",
        "com.microsoft.Teams",
        // 添加更多应用的bundle identifier
    ]
}
```
